Variables in 2D_multiresolution
p = Bspline degree (x direction, fixed to be 3 in this program)
q = Bspline degree (y direction, fixed to be 3 in this program) 
I0 = moving image (2 dimensional matrix) (will updated in each iteration and will bescaled according to multiresolution level) 
I1 = fixed image (2 dimensional matrix) (will bescaled according to multiresolution level) 
I0_new =  image after composing with given transformation field 
nlevel = number of multiresolution level (double )
max_Iteration = maximum number of iteration (double)
TOL = stopping tolarance (double)
gamma = regularize parameter used to control diffeomorphic registration (double)
PlotAll = option to plot images throughout the registration process (1 or 0 )
SaveVideo = option to save the plotted frame (1 or 0) 
UseGaussian = option to use a gaussian filter to smooth the transformation field (1 or 0) 
I00 = Original moving image (2 dimensional matrix) 
I11 = Original fixed image (2 dimensional matrix) 
M = array for computing B-spline kernel (1 x 4 array) 
M_filt = B-spline kernel used for computing coefficient (4 x 4 matrix)  
Fx = Kernel used for computing partial difference with respect to x (4 x 4 matrix)  
Fy = Kernel used for computing partial difference with respect to y (4 x 4 matrix)  
f = frame used to save the plotted images
levelcount = current multiresolution level  (double)
I0_ori= scaled original moving image (2 dimensional matrix) 
II0_ori = reshaped I0_ori (array) 
nx = size of image (x direction) (double)
ny = size of image (y direction) (double)
x_disp = displacement in x direction  (2 dimensional matrix) 
y_disp = displacement in y direction  (2 dimensional matrix) 
kl = value used to adjust displacement in x direction 
l1 = value used to adjust displacement in y direction 
x = adjusted x_disp
y= adjusted y_disp 
basis_x = matrix for computing Bspline basis value for given x 
basis_y = matrix for computing Bspline basis value for given y
coef_loc = local coefficient   for compose image with transformation field
coef_loc_x = local coefficient for compose transformation field in x direction
coef_loc_y = local coefficient for compose transformation field in y direction 
Vx = transformation field in x direction  (2 dimensional matrix) 
Vy = transformation field in y direction  (2 dimensional matrix) 
Vx_ident = identity transformation field (x direction)  (2 dimensional matrix) 
Vy_ident = identity transformation field (y direction)  (2 dimensional matrix) 
Vx_temp =  deformation value in x direction  (2 dimensional matrix) 
Vy_temp =  deformation value in y direction  (2 dimensional matrix) 
Vx_old = transformation field in x direction (2 dimensional matrix) 
Vy_old = transformation field in y direction (2 dimensional matrix) 
Vx_old_inv = inverse transformation field in x direction (2 dimensional matrix) 
Vy_old_inv = inverse transformation field in y direction (2 dimensional matrix) 
Vx_new = composed transformation field in x direction (2 dimensional matrix) 
Vy_new = composed transformation field in y direction (2 dimensional matrix) 
Vx_new_inv = composed transformation field in x direction (2 dimensional matrix) 
Vy_new_inv = composed transformation field in y direction (2 dimensional matrix) 
Vxl = transformation field (x direction)  (2 dimensional matrix) 
Vyl = transformation field (y direction)  (2 dimensional matrix) 
VXL = reshaped Vxl (array)
VYL = reshaped Vyl (array)
Vxl_inv = inverse transformation field (x direction) (2 dimensional matrix) 
Vyl_inv = inverse transformation field (y direction) (2 dimensional matrix) 
VXL_inv = reshaped Vxl_inv (array)
VYL_inv = reshaped Vyl_inv (array)
Vxl_ident = inverse transformation field (x direction)  (2 dimensional matrix) 
Vyl_ident = inverse transformation field (y direction)  (2 dimensional matrix) 
VGx = Value used for compute deformation grid in x direction (2 dimensional matrix) 
VGy = Value used for compute deformation grid in y direction (2 dimensional matrix) 
gridX0 = identity deformation grid (2 dimensional matrix) 
gridY0 = identity deformation grid (2 dimensional matrix) 
In_gridX = inverse deformation grid  in x direction (2 dimensional matrix) 
In_gridY = inverse deformation grid in y direction (2 dimensional matrix) 
FgridX = forward deformation grid in x direction (2 dimensional matrix) 
FgridY = forward deformation grid in y direction (2 dimensional matrix) 
scale = scaling factor for resizing image and transformation field (double)
coef = coefficient of level set representation of I0 (2-dimensional matrix) 
coef_I0 = coefficeint of level set representation of I0_ori(2 dimensional matrix)
CI0 = reshaped coef_I0 (array) 
TOL_Average = stopping tolerance for average relative residual (double)
InitialResidual = Initial difference between the moving and fixed image (double)
FinalResidual = difference between the registered moving image and fixed image (double)
Final_relative_residual = final relative residual (double)
sigma_par = standard deviation of the Gaussian filter (double)
smooth_par = size of the Gaussian filter (array)
Hsmooth = Gaussian filter (matrix with size (smooth_par)
I_diff = difference between registered moving image with fixed image (2-dimensional matrix) 
alpha = small value used to avoid division by zero in computing transformation field (double)
ngx = size of deformation grid in x direction (double)
ngy = size of deformation grid in y direction (double)
coef_x = coefficient of transformation field in x direction (2 dimensional matrix)  
coef_y = coefficient of transformation field in y direction (2-dimensional matrix) 
coef_x_inv =  coefficient of inverse transformation field in x direction (2 dimensional matrix) 
coef_y_inv =  coefficient of inverse transformation field in y direction (2 dimensional matrix) 
coef_x_fun = reshaped coef_x (array)
coef_y_fun = reshaped coef_y (array)
temp_coef_x =coefficient of transformation field in x direction (obtained from filter)
temp_coef_y =coefficient of transformation field in y direction (obtained from filter)
temp_coef_x_inv =coefficient of inverse transformation field in x direction (obtained from filter)
temp_coef_y_inv =coefficient of inverse transformation field in y direction (obtained from filter)
quiver_matrix_x = value used for plotting current transformation field in the x direction 
                                (2 dimensional matrix) 
quiver_matrix_y = value used for plotting current transformation field in the y direction
                               (2-dimensional matrix)
plot_pix_index_column = linespace used for plotting transformation field (array)             
plot_pix_index_row  = linespace used for plotting transformation field (array)
residualHistory = relative residual of the latest 6 iterations (array)
SR= relative residual value saved for plotting graph  (array)
SRcount = count number of relative residual for use in plotting relative residual graph (double)
didx_temp = partial difference of Image I0 with respect to x direction  (obtained by filtering coef with Fx) (2 dimensional matrix)
didy_temp = partial difference of Image I0 with respect to y direction (obtained by filtering coef with Fy) (2-dimensional matrix)
didx = resized didx_temp (2 dimensional matrix)
didy = resized didy_temp (2 dimensional matrix)
N = denominator of transformation field (double)
time_step = time step in optimization (double)
avgresidualHistory1 = average of the first three values in residualHistory 
avgresidualHistory2 = average of the last three values in residualHistory
th = subplot frame 
th1 =subplot frame
th2 =subplot frame 
th3 = subplot frame 
th4 =subplot frame 
th5 = subplot frame
